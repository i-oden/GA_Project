The files in this folder contain the code written and used during the project.
The txt-format files contain code which have been run on the command line, either locally or
on uppmax. 
The sh-format files are batch scripts which have been run on uppmax. 

File number 3 is the code used for quality checking of the RNA reads before the trimming. 
File number 5 is the code used after the trimming, but also using FastQC.
